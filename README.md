0.3

## Технологии
- JS
- TypeScript
- WebSocket
- Handlebars
- Parcel

## Установка

- `npm install` — установка стабильной версии,
- `npm run dev` — запуск версии для разработчика,
- `npm run build` — сборка стабильной версии,
- `npm run start` — сборка и запуск Node.JS сервера.
